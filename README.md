# Blog Fullstack JS

## Technos

- React Js (boilerplate: create react app)
- Node Js
- Express Js
- Mongo DB/ Mlab
- Webpack

## Installation

Cloner le projet et ouvrir 3 fenêtres dans le terminal

Fenêtre 1

### `npm start`

Fenêtre 2

### `nodemon server.js`

Fenêtre 3

### `mongo`
